### Definition of Computer Programming

A computer program consists of a sequence of instructions for a computer to perform tasks.  
Programmers must provide necessary instructions to enable computers to execute desired tasks.  
Understanding a computer's capabilities is essential for effective communication and instruction.

---

### Programming Languages

Communication with computers requires a medium, which is programming languages.  
Natural languages are easily understood by humans but not by computers, which interpret binary codes (0s and 1s).  
Compilers and interpreters serve as translators between human-readable code and machine code.

---

### Types of Programming Languages

- **Low-level programming languages** are closer to machine code and harder for humans to understand, such as Assembly language.
- **High-level programming languages** are easier for humans to understand, with examples including Java and Python.
- The necessity for multiple programming languages arises from varying requirements and contexts in programming.

---

### Programming Paradigms

#### Imperative Programming

Imperative programming involves explicitly describing a step-by-step process for task completion.

**Types of imperative programming:**

1. Procedural programming (e.g., C, Pascal)
2. Object-oriented programming (e.g., Java, C#)

> Python supports both procedural and object-oriented paradigms.

#### Declarative Programming

Declarative programming focuses on the desired outcome rather than the process to achieve it.

**Types of declarative programming:**

1. Logic programming (e.g., Prolog)
2. Markup languages (e.g., HTML) are examples of declarative programming.

### Computational Thinking

Computational thinking emphasizes problem-solving through computing techniques. Key components include:

1. **Decomposition** – Breaking down complex problems into smaller, manageable parts.
2. **Abstraction** – Ignoring irrelevant details to focus on the essential aspects of a problem.
3. **Pattern Recognition** – Identifying similarities or patterns within problems to find solutions.
4. **Algorithmic Design** – Creating step-by-step procedures to solve problems.

---

### Algorithms

An algorithm is a step-by-step solution to a computational problem.

**Key aspects of an algorithm:**

- **Input** – The data that is fed into the algorithm.
- **Output** – The result produced by the algorithm after processing the input.
- **Finite Steps** – The algorithm must have a limited number of steps to ensure it can be completed.
- **Unambiguous Steps** – Each step must be clearly defined and understandable.

A good algorithm is characterized by **efficiency**, **scalability**, and **reusability**.

### Steps to Write a Computer Program

1. Identifying the problem
2. Breaking it into manageable components
3. Choosing a programming language
4. Writing code
5. Testing and debugging the code
